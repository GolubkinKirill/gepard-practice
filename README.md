## Тема проекта

Разработка дизайна веб-приложения для ООО «Гепард».

## Назначение проекта

Веб-приложение предназначено для учета клиентских заявок. Пользователь может зарегистрироваться, войти в личный кабинет, создать заявку и просмотреть список своих заявок. Администратор может видеть все заявки и контролировать их статусы.

## Реализованные страницы

- `index.html` — главная страница;
- `login.html` — страница входа;
- `register.html` — страница регистрации;
- `dashboard.html` — личный кабинет пользователя;
- `requests.html` — список заявок;
- `admin.html` — панель администратора.
